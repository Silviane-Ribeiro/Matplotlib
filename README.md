Atividade: Visualização de Dados de Estoque

Objetivo
Esta atividade tem como objetivo praticar o uso da biblioteca **Matplotlib** para gerar diferentes tipos de gráficos em Python, representando dados de estoque de produtos.
Gráficos Desenvolvidos

1. Tendência de Estoque Diário:  
   Gráfico de linha mostrando a variação da quantidade de um produto ao longo de sete dias.

2. Comparação de Produtos:  
   Gráfico de barras comparando as quantidades em estoque de diferentes produtos (Teclado, Mouse, Monitor e Webcam).

3. Proporção de Categorias:  
   Gráfico de pizza representando a proporção do valor total de estoque por categoria (Eletrônicos, Vestuário e Alimentos).

4. Preço vs. Quantidade:  
   Gráfico de dispersão analisando a relação entre o preço unitário e a quantidade em estoque de diversos itens.

Ferramentas Utilizadas
- **Python 3**
- **Biblioteca Matplotlib**
